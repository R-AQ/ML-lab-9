# ML-lab-9

This project uses loan_data.csv to predict if a loan was fully paid or not.

I explored the data, converted the `purpose` column into dummy variables, then trained two models whichare the decision Tree
and random Forest

The random Forest had higher accuracy, but it almost always predicted class 0. Decision Tree was better at detecting class 1, which meant loans that were not fully paid.

So, Decision Tree was more useful for this dataset.
